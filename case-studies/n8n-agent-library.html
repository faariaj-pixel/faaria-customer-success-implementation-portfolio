Case study

n8n AI Agent Workflow Library

A library of foundational AI agent workflows covering tools, triggers, data transformation, and flow control.

Contact FaariaBack to portfolio

Challenge

Reliable AI automation requires more than prompts: it needs correct triggers, tool routing, memory, data shaping, and conditional logic.

Approach

  • Built/documented agent patterns for Gmail, Sheets, Docs, Calendar, Search, Calculator, Date/Time, and agent delegation.
  • Documented trigger patterns: manual, chat, schedule, and form submission.
  • Documented data transformation: field edits, filters, limits.
  • Documented flow control: IF and Switch routing.

Outputs

  • Workflow documentation
  • System prompts
  • n8n JSON workflow files
  • Routing and transformation patterns

Relevance to Customer Success & Implementation

Relevant to implementation and customer operations because automation depends on intake, retrieval, routing, action, escalation, and reporting patterns working reliably together.
