Case study

Multi-Agent Stock Analysis System

A multi-agent n8n system that combines fundamental and technical analysis using search-grounded specialist agents.

Contact FaariaBack to portfolio

Challenge

A single agent can miss important context or over-index on one perspective. Multi-agent review creates a more balanced, auditable output.

Approach

  • Designed an orchestrator agent to coordinate specialist agents.
  • Required both agents to use search before responding.
  • Separated fundamental and technical views before synthesis.
  • Returned structured bullish/bearish/neutral signals.

Outputs

  • Orchestrator Agent
  • Fundamental Analysis Agent
  • Technical Analysis Agent
  • Gemini Search tools
  • Combined final analysis format

Relevance to Customer Success & Implementation

Relevant to customer and implementation work because it shows multi-perspective review, current-data grounding, and structured synthesis for decision support.
